[ENICO VECK 게시글 작성 가이드]

1) 파일 배치 규칙
- 각 카테고리 폴더의 "썸네일용" : 메인페이지 카드에 노출할 대표 이미지 1장
- 각 카테고리 폴더의 "상세보기용" : 상세 팝업/상세 페이지에 노출할 이미지 여러 장
- 컬렉션도 동일하게 사용

2) 의류 게시글 작성 시 입력 권장값 (관리자 > New Post)
- 제목(Title): 제품명 + 시즌/라인
  예) ENICO VECK 전술 해체 베스트 26FW
- 카테고리(Category): 아우터 / 셔츠 / 팬츠 / 가방 / 인형 / 드레스
- 가격(Price): 숫자만 입력 (원 단위)
  예) 129000
- 통화(Currency): KRW
- 상세 설명(Description): 2~4문장
  예) 구조적인 패턴과 와이드 실루엣이 특징인 아이템입니다. 단독 착용과 레이어드 모두 안정적인 핏을 제공합니다.
- 의류 사양(Specs): 줄바꿈으로 항목 구분
  예)
  소재: 코튼 100%
  핏: 오버핏
  두께감: 보통
  신축성: 약간
  세탁: 찬물 단독세탁 권장
- 이미지(Images):
  1) 썸네일용 폴더 이미지 1장 먼저 추가
  2) 상세보기용 이미지를 그 다음 순서로 추가
- 공개 여부: ON (노출), OFF (임시저장)

3) 컬렉션 게시글 작성 시 입력 권장값 (관리자 > Collections)
- 제목(Title): 컬렉션 이름
  예) RUNWAY 26FW
- 시즌(Season): 26FW / 27SS 형식 권장
- 설명(Description): 카드에서 보일 짧은 설명 1~2문장
- 상세 설명(Full Description): 컬렉션 컨셉 설명 2~5문장
- 출시일(Release Date): YYYY-MM-DD
  예) 2026-03-15
- 항목 수(Items): 이미지/룩 수
- 이미지(Images):
  1) 썸네일용 폴더 이미지 1장 먼저 추가 (메인 썸네일)
  2) 상세보기용 이미지 전부 추가
- 공개 여부: ON

4) 업로드 전 체크리스트
- 대표 썸네일은 반드시 1장 선정
- 상세 이미지 순서를 룩북 흐름대로 정렬
- 설명/사양에 오탈자 확인
- 가격/통화/KRW 확인
